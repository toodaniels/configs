# Configs

## This configuration is tested in the following:

- Manjaro=5.15.21-1
- Arch=5.16.9-arch1-1

## Clone repo
```
    git clone https://github.com/toodaniels/configs.git
    cd configs
```
## Install Arch packages

```
    chmod +x install.sh
    ./install.sh
```

## Add config files

```
    chmod +x config-files.sh
    ./config-files.sh
```
